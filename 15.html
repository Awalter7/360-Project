TEACHER’S ASSISTANT DATABASE UNIVERSITY DATABASE MANAGEMENT SYSTEM What is the Teacher’s Assistant Database? The Teacher’s Assistant (TA) Database is a lisng of TAs assisng Professors at an instuon of higher learning. Each Teacher’s Assistant is idenﬁed by their ﬁrst and last name, date of birth, university ID, and department. Each of the aforemenoned aributes associated with Teacher’s Assistant may be managed through this interface. The corresponding data input into the applicaon for each Teacher’s Assistant is stored in a text ﬁle, TeacherAssistant.txt. Each row in the ﬁle represents a diﬀerent TA. Each column per individual row represents informaon (aributes) pertaining to a single TA, for example: name, departments, courses assisng, etc. In order to add, delete, or make modiﬁcaons to Teacher’s Assistants in the database, you must choose the Teacher’s Assistant Admin user role.
